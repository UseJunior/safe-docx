# Safe DOCX Suite

Monorepo for the Safe DOCX package family.

## Packages

- `@usejunior/docx-primitives`
- `@usejunior/docx-comparison`
- `@usejunior/safedocx`
- `safedocx` (unscoped alias package)
- `@usejunior/safedocx-mcpb` (private MCP bundle wrapper)

## Development

```bash
npm ci
npm run build
npm run test:run
npm run check:spec-coverage
```
