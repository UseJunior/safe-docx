# OpenSpec Template Spec

This folder is an intentional starter template for new OpenSpec changes.

Usage:
1. Copy `openspec/templates/template-spec` to `openspec/changes/<change-id>`.
2. Rename `<change-id>` to a unique verb-led ID (for example `add-...`, `update-...`, `remove-...`).
3. Replace placeholder text in `proposal.md`, `tasks.md`, and `specs/*/spec.md`.
4. Run `openspec validate <change-id> --strict`.
