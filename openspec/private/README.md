# Private OpenSpec Drafts

Use this folder for local brainstorming, rough notes, and speculative proposals that are not ready for public review.

Guidelines:
- Keep publishable, reviewable proposals under `openspec/changes/<change-id>/`.
- Keep only non-committable drafts under `openspec/private/`.
- Promote a draft by copying it into a real change folder with proper `proposal.md`, `tasks.md`, and spec deltas.
