# Code of Conduct

By participating in this project, you agree to act responsibly and respectfully.

## Expected Behavior

- Be respectful in technical disagreement.
- Focus feedback on code and behavior, not people.
- Help keep discussions constructive and specific.

## Unacceptable Behavior

- Harassment, intimidation, or abuse.
- Deliberate deception, fraud, or malicious use.
- Publishing private/confidential information without permission.
- Any illegal or harmful activity.

## Enforcement

Maintainers may remove content, restrict participation, or block contributors for violations.

## Reporting

Report concerns to `steven@usejunior.com`.
